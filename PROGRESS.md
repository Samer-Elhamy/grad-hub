# Grad Hub — Progress

**الإصدار الحالي:** v3 — React + Express + Arabic/English UX + feedback loop.

الواجهة الأساسية الآن في `web/`، والـ API في `backend/`. نسخة `site/` أصبحت prototype/legacy static.

## v3 Highlights

- واجهة bilingual: عربي RTL وإنجليزي LTR.
- Discover swipe cards بصور مرتبطة بكل فكرة.
- Heart وStar مستقلين في history والـ analytics.
- Preferences summary يتحدث من history الحقيقي.
- Top Keywords تتغير حسب swipes.
- صفحة تفاصيل لكل فكرة فيها تحليل نقدي ورابط مصدر خارجي.
- README تسويقي وسكرينشوتات وصورة preview للريبو.

## المميزات الجديدة

### 🌙 دارك مود
- زر تبديل في الهيدر (☀/🌙)
- يحفظ التفضيل في `localStorage`
- ألوان مستوحاة من GitHub Dark (`#0d1117`)

### 🧠 الوكيل الناقض (Critic Agent)
- **يحلل كل فكرة** ويعطي: نقاط القوة، نقاط الضعف، اقتراحات التطوير
- **نظام تسجيل:** ٤ محاور (الجدوى، الابتكار، الطلب، القيمة التعليمية)
- **توصيات مخصصة** حسب تفضيلاتك
- **مقارنة** بين فكرتين

### 🔄 حلقة التغذية الراجعة
- **تقييم** (١-٥ نجوم) + تعليق لكل فكرة
- **تتبع تلقائي** للفئات والكلمات المفتاحية التي تحبها
- **تحسين البحث:** الفئات التي تحصل على تقييم عالٍ تظهر أكثر
- **تصدير السجل** إلى JSON

### 📋 هيكل المشروع الجديد
```
site/
├── index.html          ← لوحة المراجعة (فكرة واحدة كل مرة)
├── preferences.html    ← تفضيلاتي
├── history.html        ← سجلي
├── ideas.html          ← مستعرض المشاريع (احتياطي)
├── feedback.html       ← نموذج تعليقات عام
├── critic.js           ← الوكيل الناقض
├── app.js              ← المتحكم الرئيسي
└── css/style.css       ← دارك مود + RTL + responsive
```
